# signers
Сущность Signer описывает подписанта (Имя, Отчество, Фамилия, Организация, Дата)

У сущности Signer есть слушатель SignerEventListener, который перед вставкой и обновлением сущности в БД шифрует ее поля.

Создана пользовательская роль Decrypt Reader, которая позволяет видеть расшифрованый текст и редактировать и создавать сущности.

В окне SignerBrowser для ролей Decrypt Reader и Administrator, после отображения страницы объекты в контейнере CollectionLoader'а
расшифровываются и таким образом отображается расшифрованый текст.

В окне SignerEdit  для ролей Decrypt Reader и Administrator, после отображения страницы, при наличии объекта, расшифровываются 
поля объекта. При сохрании объекта, вызывается слушатель сущности, шифрует объект и вставляет/обновляет его в БД. 
Перед закрытием окна сущность в InstanceContainer расшифровывается, и в SignerBrowser возвращается уже расшифрованый тескт.
